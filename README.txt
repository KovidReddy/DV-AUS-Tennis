The data files are data_all and data1 to data2, and also the .csv file (where the percentages have been filtered)

Please run on Firefox for best results, if you would want to run in Chrome start a local server by following below steps:

1. Open Command Prompt or Terminal
2. Navigate to root folder using cd
3. type in -m SimpleHTTPSever 8000
4. Open chrome using localhost:8000
